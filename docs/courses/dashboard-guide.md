# Course Dashboard Guide

## Two Dashboards, Two Roles

Teezy provides two separate dashboards for course staff — one for day-to-day tee sheet management, and one for owners and managers who need a broader view. Each has its own login credentials.

---

## Pro Shop Dashboard

**Who uses it:** Pro shop staff responsible for daily tee time operations.

### What You Can Do

| Feature | Description |
|---------|-------------|
| **Tee sheet management** | View and manage daily tee time slots |
| **Upcoming bookings** | See who's booked: player name, party size, selected game mode |
| **Availability control** | Mark slots as available or unavailable in real time |
| **Check-ins** | View today's check-ins as players arrive |

### What You Can't Do

The Pro Shop Dashboard is intentionally limited to operational tasks. You cannot access revenue data, edit the course profile, create events, or manage billing. Those functions belong to the Manager Dashboard.

---

## Manager / Owner Dashboard

**Who uses it:** Course owners, general managers, and directors of golf.

### What You Can Do

#### Revenue & Billing
- View bookings this month and projected monthly fee
- Download invoices and billing history
- Update payment details via Stripe

#### Course Profile
- Edit course name, description, and photos
- Manage location settings

#### Mood Tags
Assign mood tags that describe your course's vibe. These appear in the player discovery feed and help golfers find you based on what they're looking for. Available tags include:
- Competitive
- Relaxed
- Beginner-Friendly
- Fast Pace
- Social
- Advanced
- Scenic

Good mood tags = better match rates with the right golfers.

#### Events & Promotions
Create events that appear in the player discovery feed and social feed:
- Ladies Night, Couples Night, Happy Hour, etc.
- Set date, description, and optional image
- Events are automatically shown to nearby golfers

See the [Event Creation Guide](./event-creation-guide.md) for step-by-step instructions.

#### Mini Tournaments
Set up and manage mini tournaments for your course. See the [Tournament Creation Guide](./tournament-creation-guide.md) for full details.

#### Settings
- Manage staff accounts (add/remove Pro Shop and Manager logins)
- Update notification preferences
- Configure course-level privacy settings

## Frequently Asked Questions

**Can Pro Shop staff be upgraded to Manager access?**
Yes. From the Manager Dashboard under Settings, you can grant elevated permissions to any staff account.

**Can I have multiple Manager accounts?**
Yes. You can add multiple manager-level accounts for different team members (e.g., owner + GM both have full access).

**Are the two dashboards separate apps or the same app?**
They are the same app — the dashboard your staff sees is determined by their account type and login credentials. Pro shop staff see the Pro Shop view; managers see the full Manager view.

**Can I see real-time booking notifications?**
Yes. Both dashboards show live updates as bookings come in. Pro Shop staff get notified of new bookings affecting the current day's tee sheet.
